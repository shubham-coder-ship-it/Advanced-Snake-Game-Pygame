# 🐍 Advanced Snake Game (Pygame)

An advanced Snake Game built using **Python** and **Pygame** featuring multiple levels, lives, high score tracking, themes, sound effects, settings menu, and mobile-style controls.

---

# ✨ Features

- 🎮 Interactive Main Menu
- ❤️ Three Lives System
- 🏆 High Score Saving
- 📈 Level Progression
- 🔊 Sound ON/OFF
- 🎨 Multiple Themes
  - Classic
  - Ocean
  - Forest
  - Night
- ⚙️ Settings Menu
- 📱 Mobile-style On-screen Controls
- ⏸ Pause & Resume
- 🔁 Restart Option
- 🍎 Random Food Generation
- 💥 Wall & Self Collision Detection
- 🚀 Speed Increases Every Level
- 🎵 Eating Sound Effect

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/shubham-coder-ship-it/Advanced-Snake-Game-Pygame.git
```

### Go to the project folder

```bash
cd Advanced-Snake-Game-Pygame
```

### Install Pygame

```bash
pip install pygame
```

### Run the game

```bash
python snake_game.py
```
